Krypto-Typ

  • Sie können den voreingestellten Krypto-Typ (welcher der PHP-MD5-Funktion entspricht) benutzen, oder Sie geben org.apache.openmeetings.utils.crypt.MD5CryptImplementation ein, um BSD-artige Verschlüsselung zu benutzen. Für weitergehende Informationen, auch zum Schreiben einer eigenen Krypto-Klasse, siehe Custom Crypt Mechanism. Sie können diesen Wert später verändern, ABER vorher erstellte Nutzer und Sitzungen könnten anschließend unbenutzbar sein.
